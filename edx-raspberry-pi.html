edX on the Raspberry Pi

Wed 01 January 2014
By jarv

edx.org can now be run on the raspberry pi! Using Ansible and the same configuration scripts we use in production it's pretty straightforward to setup. More details on the edX engineering blog post.

Proudly powered by Pelican, which takes great advantage of Python.
